渐变属性说明

所有6个面都使用相同的渐变方向：to bottom（从上到下）

前面 (front)
background: linear-gradient(to bottom, #ff3333, #33cc33);
后面 (back)
background: linear-gradient(to bottom, #ff3333, #33cc33);
右面 (right)
background: linear-gradient(to bottom, #ff3333, #33cc33);
左面 (left)
background: linear-gradient(to bottom, #ff3333, #33cc33);
上面 (top)
background: linear-gradient(to bottom, #ff3333, #33cc33);
下面 (bottom)
background: linear-gradient(to bottom, #ff3333, #33cc33);

Transform属性说明

前面 (front)
transform: translateZ(100px);
后面 (back)
transform: translateZ(-100px) rotateY(180deg);
右面 (right)
transform: skewX(-5deg) translateX(100px) rotateY(90deg);
左面 (left)
transform: skewX(5deg) translateX(-100px) rotateY(-90deg);
上面 (top)
transform: skewY(5deg) translateY(-100px) rotateX(90deg);
下面 (bottom)
transform: skewY(-5deg) translateY(100px) rotateX(-90deg);

注：所有面都使用完全相同的线性渐变，方向为"to bottom"，颜色从红色(#ff3333)渐变到绿色(#33cc33)。
